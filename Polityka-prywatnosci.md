Polityka Prywatności
Data wejścia w życie: 9 grudnia 2025

1. Informacje ogólne
Niniejsza Polityka Prywatności opisuje zasady korzystania z aplikacji Ignaś Memory Game (dalej: „Aplikacja”).
Aplikacja jest prostą grą typu memory, przeznaczoną dla użytkowników w każdym wieku, w tym dla dzieci poniżej 13 roku życia.

2. Brak zbierania danych osobowych
Aplikacja nie zbiera, nie przetwarza ani nie przechowuje żadnych danych osobowych użytkowników.

W szczególności:
Aplikacja nie wymaga zakładania konta,
Aplikacja nie gromadzi danych identyfikujących użytkownika,
Aplikacja nie zbiera danych kontaktowych, lokalizacyjnych, ani technicznych,
Aplikacja nie korzysta z plików cookies ani podobnych mechanizmów śledzących.

3. Wprowadzanie imienia lub nazwy gracza
Aplikacja umożliwia użytkownikowi wpisanie imienia lub dowolnego tekstu identyfikującego gracza wyłącznie na potrzeby bieżącej rozgrywki.

Informacja ta:
jest wykorzystywana tylko lokalnie i tymczasowo,
nie jest zapisywana ani przechowywana po zamknięciu Aplikacji,
nie jest przesyłana nigdzie poza urządzenie użytkownika,
po ponownym uruchomieniu Aplikacji musi zostać wpisana ponownie.

4. Dzieci
Aplikacja jest bezpieczna dla dzieci i nie zbiera żadnych danych od dzieci, w tym danych osobowych w rozumieniu obowiązujących przepisów (np. RODO, COPPA).
Nie są wymagane żadne zgody rodzicielskie, ponieważ Aplikacja nie przetwarza danych osobowych.

5. Udostępnianie danych podmiotom trzecim
Aplikacja:
nie udostępnia danych podmiotom trzecim,
nie korzysta z zewnętrznych narzędzi analitycznych,
nie integruje się z usługami zewnętrznymi wymagającymi przekazywania danych.

6. Zmiany Polityki Prywatności
Treść niniejszej Polityki Prywatności może zostać zaktualizowana w przypadku zmian w Aplikacji. Aktualna wersja Polityki Prywatności będzie zawsze dostępna wraz z Aplikacją.

7. Kontakt

Privacy Policy
Effective Date: December 9, 2025

1. General Information
This Privacy Policy describes the terms of use of the Ignaś Memory Game application (hereinafter referred to as the "Application").
The Application is a simple memory game intended for users of all ages, including children under 13.

2. No Collection of Personal Data
The Application does not collect, process, or store any personal data from users.

In particular:
The Application does not require the creation of an account,
The Application does not collect data identifying the user,
The Application does not collect contact, location, or technical data,
The Application does not use cookies or similar tracking mechanisms.

3. Entering a Player Name or Username
The Application allows the user to enter a name or any text identifying the player solely for the purposes of the current game.

This information:
is used only locally and temporarily,
is not saved or stored after closing the Application,
is not transferred anywhere outside the user's device,
must be re-entered after restarting the Application.

4. Children
The Application is safe for children and does not collect any data from them, including personal data as defined by applicable regulations (e.g., GDPR, COPPA).
No parental consent is required because the Application does not process personal data.

5. Sharing Data with Third Parties
The Application:
does not share data with third parties,
does not use external analytical tools,
does not integrate with external services that require data transfer.

6. Changes to the Privacy Policy
This Privacy Policy may be updated in the event of changes to the Application. The current version of the Privacy Policy will always be available with the Application.

7. Contact
If you have any questions regarding this Privacy Policy, please contact the developer of the Application:
